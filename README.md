## Description

Based on the example node server from [godot-demo-projects](https://github.com/godotengine/godot-demo-projects/tree/master), 
specifically the _networking/webrtc_signaling_ example.

## Prerequisites
- [npm](https://www.npmjs.com/)
- [Typescript](https://www.typescriptlang.org/)

## How to run
`npm run dev`
